A desktop application developed for the Visual Programming course at John von Neumann University. The system manages vehicle repair
records, logging customer vehicles, tracking repair jobs, and displaying service history built using C and C++.
Features
• Register vehicles and customer information
• Log and track repair jobs with status updates
• View service history per vehicle
• Basic CRUD operations (Create, Read, Update, Delete) for repair records
Tech Stack
Languages: C, C++
Course: Visual Programming — John von Neumann University
Paradigm: Structured and Object-Oriented Programming
Getting Started
 git clone https://github.com/JoviMs/Vehicle-Repair-System.git
 cd Vehicle-Repair-System
Compile using your C/C++ compiler (e.g. GCC or the IDE used in class):
 g++ -o VehicleRepair main.cpp
 ./ VehicleRepair
